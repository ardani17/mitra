Test Desktop Sync Button

To test the desktop sync button:

  1. Open your browser and go to: http://localhost:8000/projects/28 (or your project URL)
  2. Open Browser Developer Console (F12)
  3. Click on the "Cek Sinkronisasi" button
  4. Check the console for any errors

Expected Behavior:

  • Button should show loading spinner while checking
  • If synced: Button turns green with "Tersinkronisasi"
  • If not synced: Button turns orange with "Sinkronkan (X)" where X is the number of issues
  • Alert should show summary of issues if any found

Common Issues:

  • Button not responding: Check console for JavaScript errors
  • 404 Error: API endpoint not found - check routes
  • 500 Error: Server error - check Laravel logs

Manual API Test:

You can also test the API directly in browser console:

// Test check sync API
fetch('/api/file-explorer/project/28/check-sync', {
    headers: {
        'Accept': 'application/json',
        'X-CSRF-TOKEN': document.querySelector('meta[name="csrf-token"]').content
    }
})
.then(response => response.json())
.then(data => console.log('Sync Status:', data))
.catch(error => console.error('Error:', error));
